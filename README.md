# 🏥 Patient Record Management System

## 📌 Overview
This project implements a **Patient Record Management System** in Python using multiple data structures. It demonstrates how different structures handle storing, searching, deleting, and traversing patient records.

---

## 🧠 Data Structures Used
- Dynamic Array (List)
- Linked List
- Hash Map (Dictionary)
- Priority Queue

All structures follow a common interface:  
`insert`, `search`, `delete`, and `traverse`.

---

## 🧾 Patient Data
Each record includes:
- ID, Name, Age, Gender  
- Medical Condition  
- Hospital  
- Insurance Provider  
- Billing Amount
- Admission Type  

---

## ⚙️ Features
- Compare performance of different data structures  
- Load patient data from CSV  
- Benchmark insert, search, delete, and traverse operations
